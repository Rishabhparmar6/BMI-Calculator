# 🧮 BMI Calculator Web App

## 📌 Description

This is a responsive and interactive **BMI (Body Mass Index) Calculator** built using HTML, CSS, and JavaScript. It allows users to input their height and weight in different units and instantly calculates their BMI along with the corresponding health category and suggestions.

This project demonstrates practical knowledge of frontend development and user interaction.

---

## 🚀 Features

* 📏 Supports multiple units (cm, meters, inches, kg, pounds)
* ⚡ Real-time BMI calculation
* 📊 Displays BMI value with category (Underweight, Normal, Overweight, Obese)
* 💡 Provides health suggestions based on result
* 🔄 Reset functionality
* 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## ⚙️ How It Works

* User enters height and weight

* Inputs are converted into standard units

* BMI is calculated using formula:

  **BMI = weight (kg) / height² (m²)**

* Based on BMI, category is shown with message
